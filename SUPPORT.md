# Support

This is a pre-release research and engineering project without a guaranteed
support SLA.

- Search existing [GitHub Issues](https://github.com/east-true/adaptive-ai-orchestrator/issues)
  before opening a new one.
- Use the matching bug, feature-request, or question form for public,
  non-sensitive requests.
- Include the project version or commit, Python version, operating system,
  command, expected result, and a minimal reproduction when reporting a bug.
- Remove tokens, prompts containing private data, home-directory paths, and
  local execution logs before posting.
- Follow [SECURITY.md](SECURITY.md) for vulnerabilities or accidentally exposed
  credentials; never disclose those details in a public issue.

The maintainers cannot provide private consulting or debug third-party agent
accounts, subscription limits, or provider outages.
